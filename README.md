# SoundScout
SoundScout – Discover music smarter, faster, better.

Project Overview

SoundScout is a web application that allows users to search and explore music tracks using a public API. The application provides features like searching, filtering, and sorting songs, along with an interactive and responsive user interface.

Objective
The goal of this project is to demonstrate the use of JavaScript, API integration, and modern UI development techniques.

API Used

This project uses the iTunes Search API to fetch real-time music data.
Endpoint:
https://itunes.apple.com/search

Features (Planned)
Search songs by name or artist
Filter songs based on artist
Sort songs (alphabetically, by release date)
Add songs to favorites
Dark/Light mode toggle
Responsive design for all devices

JavaScript
